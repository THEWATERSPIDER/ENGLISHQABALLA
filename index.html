How Counting Well Works

  1. Calculate the ALW value of the first word
  2. Multiply that value by the number of letters in the second word
  3. Calculate the ALW value of the second word
  4. Multiply that value by the number of letters in the first word
  5. Add both multiplication results together for the final Count Well total

About Simple ALW Calculation

The Simple ALW Calculator computes the gematria value of any word or phrase using the English Qaballa ALW cipher. Each letter is assigned a specific numerical value, and the total is the sum of all letter values.


The calculator ignores spaces and punctuation, focusing only on alphabetical characters. It provides a detailed breakdown showing each letter and its corresponding value.

About The Wakanaba Kamea Extrapolator

This calculator creates a magical kamea (grid) from two words, where the first word forms the vertical axis and the second word forms the horizontal axis.

Each cell in the grid contains the sum of the ALW values of the corresponding letters. These numbers are then converted into barbarous words using the WAKANABA table.

Rules for Barbarous Words:

  1. Consonants are followed by the vowel on their corresponding line (e.g., W=WA, H=HO)
  2. Vowels are followed by L (e.g., A=AL, E=EL)
  3. Numbers above 26 are split into tens and ones (e.g., 45=HO-SU)
  4. The number 2 always becomes LA
  5. Double numbers are halved (e.g., 44=HO, 55=SU)
  6. Zero is treated as 7 (e.g., 40=HO-OL)

Grand Word of the Kamea:

The Grand Word is calculated by summing all values in the kamea and converting the total to a barbarous word using the same rules. This word represents the unified magical essence of the entire kamea.
